Bucătăria tradițională românească este denumirea pe care o poartă rezultatul sintetizării, în timp, a gusturilor și obiceiurilor gastronomice specifice poporului român. Ea este diversă, cuprinde nenumărate obiceiuri și tradiții culinare, mâncăruri specifice, împreună cu obiceiuri provenite prin intersectarea culturii gastronomice cu tradiții ale altor popoare.

Tocănița de pui

Tocanița de pui este o mâncare des întâlnită în rândul țăranilor care prin creșterea găinilor și a puilor de casă le este la îndemână să o gătească.

Alături de această tocaniță este normal să o însoțească mămăliga, fără de care și-ar pierde farmecul. Mămăliga la rândul ei este un produs obținut tot de casă, din porumb (cucuruz) măcinat la moară.
